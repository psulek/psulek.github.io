This field defines the primary language in your project. Each project requires at least one language. After you add more languages, you can change primary language to a different one.

Purpose: The primary language has the following characteristics:

  • It is displayed as the first language in the Resource pane.
  • It is always exported as a reference for translations to other languages.
  • Cannot be unchecked in the Language dialog. To do so, you have to select another language as primary.
