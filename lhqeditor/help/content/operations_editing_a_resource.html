Operations Editing a Resource

You can perform the following operations on a resource in the resource pane:

  • Rename a resource
  • Edit resource description
  • Edit resource parameters
  • Change resource state
  • Edit resource translation
  • Lock / unlock resource translation
  • Translate resource using external service


Figure: Editing resource user interface

Rename a resource

To rename a resource, edit the text in the Name field.

Note: The edited resource name must be unique within the folder in the navigation tree where the resource is located.

Name is used to generate classes.

Edit resource description

To edit a resource description, edit the text in the Description field.

Description is an additional information related to the text resource, e.g. a short info where it is used or a note for translators or developers.

Edit resource parameters

To edit resource parameters, click Define button. Resource Parameters window opens.

You can perform the following operations on resource parameters:

  • Add
  • Edit
  • Remove
  • Move up
  • Move down

Change resource state

To change a resource state, select the required value in the State dropdown.


Figure: Change resource state

Note: When the resource state is Final, it is not possible to edit translations (they are locked). To enable editing again, unlock the translations.

Note: Resources in the navigation tree are colored according to their states.

Resource state is used to have a better overview which resources in your project are already translated, which of them require review, etc.

Edit resource translation

To edit a resource translation, edit the text next to the Language label in the Translations area.

Lock / unlock resource translation

To lock or unlock a resource translation for a specific language, click on ... button next to the translation text and then click on respective menu item.


Figure: Lock resource translation

Translations actions

You can perform the following actions on resource translations:

  • Translate empty values
  • Translate all values
  • Lock all values
  • Unlock all values

Translate resource using external service

To translate a resource using external service, click on ... button next to the translation text and then click on Translate [Language] Value menu item. The resource is translated automatically.


Figure: Automatically translated resource

Note: Automatically translated resource is marked by icon. After you modify the translated text manually, the icon is no longer present.

Note: In order to use external translation service, it must be configured first. When the service is not configured, Translation Service window appears. Click Yes to enable translation service. The Translator [configuration] window appears. See also: Enabling Translator Integration [operation]. Once the translation service is configured correctly, you can use the Translate [Language] Value menu item.
