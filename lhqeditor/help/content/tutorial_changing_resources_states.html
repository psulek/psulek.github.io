Tutorial Changing Resources States

Select the HelloWorld resource and change its state to Review Required:

Note that the resource in the Navigation tree has blue icon, indicating the respective state:

Save the project.

Go to Help → Preferences. Configuration window appears. Make sure General tab is selected. Check Lock translations... checkbox.

Click OK to close the window.

Change the state of the HelloWorld resource to Final.

Note that resource translations are now locked. If need be, you can unlock them (see screenshot above).
