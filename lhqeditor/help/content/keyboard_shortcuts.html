Keyboard Shortcuts

The following keyboard shortcuts are available:

Alt + 1 Focus Navigation tree
Ctrl + S Save Project and generate classes
Ctrl + Shift + S Save All and generate classes
Ctrl + Insert Create New Category (when Navigation tree is focused)
Insert Create New Resource (when Navigation tree is focused)
F2 Rename selected element in the Navigation tree (resource, category or project)
Delete Delete selected element(s) in the Navigation tree (resource, category or project)
Ctrl + X Cut selected element(s) in the Navigation tree (resource, category or project)
Ctrl + C Copy selected element(s) in the Navigation tree (resource, category or project)
Ctrl + Shift + C Copy the selected resource as WPF XAML markup expression (nuget package ScaleHQ.WPF.LHQ is required to be installed in the project)
Ctrl + Alt + C Copy the selected resource as C# code expression
Ctrl + V Paste the Navigation tree element(s) which were previously cut or copied
Ctrl + D Duplicate selected element in the Navigation tree
Ctrl + Z / Ctrl + Y Undo / redo the last operation
Ctrl + Shift + E Show Export Resources window
Ctrl + Shift + I Show Import Resources window.
Ctrl + F Focus Search panel in the Navigation tree
