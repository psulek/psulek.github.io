Installation Procedure

To install Localization HQ Visual Studio extension, proceed as follows:

  1. Double-click the Localization HQ VSIX file. VSIX Installer window appears.


  2. Click Install. The installation process starts.


  3. Once the installation finishes, click Close.


  4. Start Visual Studio. If you run the Localization HQ extension for the first time, a dialog is displayed. Click Yes to open General [configuration] window.


You can now create a new project, using an LHQ template, see Creating a New Project [tutorial]
