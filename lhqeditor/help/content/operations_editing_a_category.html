Operations Editing a Category

You can perform the following operations on a category in the resource pane:

  • Rename a category
  • Edit category description
  • Export resources to a file


Figure: Editing category user interface

Rename a category

To rename a category, edit the text in the Name field.

Note: The edited category name must be unique within the folder in the navigation tree where the category is located.

Name is used to generate classes.

Edit category description

To edit a category description, edit the text in the Description field.

Description is an additional information related to the text resource category, e.g. a short info where it is used or a note for translators or developers.

Export resources to a file

To export all resources in the selected category to a file, click Actions → Export Resources to a File menu item. Export Resources to a File dialog appears.
