Operations Editing a Project

You can perform the following operations on a project in the resource pane:

  • Rename a project
  • Edit project description
  • Export resources to a file


Figure: Editing project user interface

Rename a project

To rename a project, edit the text in the Name field.

Name is used to generate classes.

Edit project description

To edit a project description, edit the text in the Description field.

Description is an additional information related to the project, e.g. a short info how it is used or a note for translators or developers.

Export resources to a file

To export all resources in the project to a file, click Actions → Export Resources to a File menu item. Export Resources to a File dialog appears.
