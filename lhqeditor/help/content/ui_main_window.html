User Interface Main Window

The main window user interface contains the following components:

  • Toolbar
  • Navigation tree
  • Resource pane
  • Status bar
  • Information pane


Figure: Localization HQ extension main window user interface components (without information pane)

Toolbar

The toolbar contains buttons which serve as a shortcut to the following operations and settings:

  • New category
    Creates a new category under the selected item in the navigation tree. See also: Creating a New Category [operation].
  • New resource
    Creates a new resource under the selected item in the navigation tree. See also: Creating a New Resource [operation].
  • Delete
    Deletes the selected object(s) in the navigation tree. See also: Deleting an Object [operation].
  • Languages
    Opens the Language Settings [user interface] window. Displays information about languages: default language, total number of languages in the project.
  • Import
    Opens the Import Resources [user interface] window. See also: Importing Resources [operation].
  • Export
    Opens the Export Resources [user interface] window. See also: Exporting Resources [operation].
  • Properties
    Opens the Project Settings [user interface] window.
  • Plugins
    Opens the Plugins Configuration [user interface] window.
  • Help
    Displays the following additional menu items:
    • What's New? - Opens the What's New information dialog containing a list of new features in the current version.
    • License - Opens the License information dialog.
    • Preferences - Opens the General Configuration [user interface] window.
    • Online Manual - Opens this web help.
    • Product Web Page - Opens the www.lhqeditor.com web page.
    • About - Opens the About information dialog.

Navigation Tree

The navigation tree contains categories and resources. You can search for specific resources using criteria defined in the filter. You can access all resources in the project using the navigation tree UI component.

The root item in the navigation tree is marked by icon and represents the project name.

Resource Pane

The resource pane is used to edit individual resources and translations. You can edit the name of the resource, change its state, define resource parameters, edit translations and perform actions, e.g. locking translations, translate the resources using external services.

Status Bar

The status bar displays path of the selected resource in the navigation tree and the code generator which is associated with the current lhq file.

Information Pane

The information pane is located above the status bar. You can turn it off by clicking its Close icon.


Figure: Main window information pane

You can turn the information pane on again by changing the Show Hints checkbox value, see General [configuration].
