Available importers

Select one of the available importers to define how the imported files will be processed. At the moment, the following importers are available:

  • NET resources (*.resx) files importer
  • Microsoft Excel (*.xlsx) resources importer

Files to import

Select files containing resources related to the specified importer, and click OK button to start the import.

Allow import of new languages

When checked, all languages which exist in the imported file and are not in the current localization project, will be added to the project.

Default value: Checked

Import mode

The import mode defines, where the imported resources have to be located within the navigation tree.

  • Import into new category - Categories and resources imported from file(s) will be added into a new category under project root. The category name will contain current timestamp. Example:


    • Figure: Imported resources folder containing the current timestamp

  • Merge with existing resources - Categories and resources imported from file(s) will be merged with existing categories and resources. Existing categories and resources will be overwritten.

Note: Imported resources are added to the project under a new category (if tree project structure is allowed, see Project Settings [user interface]). Otherwise, imported resources are added at the bottom of the flat list.
