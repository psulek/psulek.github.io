Figure: Configuration pane, Plugins tab user interface

Installed plugins

You can enable / disable the following plugins:

  • Default Exporters/Importers Pack - When checked, exporting and importing functionalities are available.
  • Microsoft Translator - When checked, Microsoft Translator services can be configured and used.
  • Yandex Translations - When checked, Yandex Translations services can be configured and used.

Default values: Checked
