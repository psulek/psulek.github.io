Operations Exporting Resources

Exporting project resources

To export all project resources, click File → Export Resources menu item. Export Resources window opens (see Export Resources [user interface]).


Figure: Export Resources window user interface components

Proceed as follows:

  1. Select the exporter.
  2. Select the export file.
  3. Check languages you want to export.
  4. Click OK. Export starts.

After the resources are exported, success message appears and the exported file is opened by its default application, e.g. Microsoft Excel.


Figure: Resources exported successfully

Exporting selected resources

To export selected resources in the navigation tree, click Actions in the Resource pane. Menu appears. Click Export resources to file menu item. A dialog appears.
