User Interface Language Settings

The window contains a table of languages used in the project. The table is sorted by language code. One language is marked as primary.

Buttons:

  • Add - Allows to add a new language to the project. Opens the Language dialog.
  • Edit - Allows to edit language parameters. Opens the Language dialog. See description below.
  • Remove - Deletes the selected language and associated translations from the project.

Language dialog

The dialog offers the following parameters:

  • Language dropdown - Allows to select the specific language.
  • Set as primary language checkbox
  • Country specific checkbox
