Technical skills

  • Translator / resources manager - Ability to edit, translate and manage text resources.
  • Developer - Ability to implement generated code and classes in your project.

In order to install Localization HQ extension on Windows operating system, the following preconditions have to be met:

Minimum hardware requirements

Localization HQ hardware requirements are equivalent to the requirements for Visual Studio and common requirements for Windows operating system.

Software requirements for Windows operating system

  • Visual Studio 2015 / 2017 / 2019 (any edition)
