Operations Enabling Translator Integration

Configuring Translator Plugins

Note: In this topic, both Microsoft and Yandex translation services are configured.

To enable translator integration, proceed as follows:

  1. Click Plugins button on toolbar. Visual Studio Options window opens, Localization HQ menu item is selected, Plugins tab is selected.
  2. Select Translator tab.
  3. Check Enable translator integration checkbox.
  4. Select Microsoft Translator (using Cognitive Services) from dropdown.


    5. Figure: Enabling translator integration

    Note: Before you can use translation service, you have to configure it first.

  5. Click Configure button. The Configure Plugin window opens.
  6. Add azure cognitive services key to the Subscription Key text field.


    7. Figure: Configuring Microsoft translation service

  7. Click OK. The Configure Plugin window closes.

    8. Note: Once the subscription key is entered, the Test button is available to verify that the key and the service are working as expected.


    Figure: Test button appears

  8. Click Test button. Microsoft translation services test process starts, which should result in displaying successful test message.


    9. Figure: Microsoft translation service test success message

  9. Select Yandex Translator from dropdown and click Configure button. The Configure Plugin window opens.


    10. Figure: Configuring Yandex translation service

  10. Click OK. The Configure Plugin window closes.
  11. Click Test button. Yandex translation services test process starts, which should result in displaying successful test message.


    12. Figure: Yandex translation service test success message

  12. Click OK to close the Preferences window. Now you can translate resources automatically, using the selected configured service. For more information, see Editing a Resource [operation].
