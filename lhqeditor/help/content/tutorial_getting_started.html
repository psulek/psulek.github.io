Tutorial Getting Started

To start Localization HQ extension, double-click on Strings.lhq file in your project. LHQ main window appears, which should like the example below:


Figure: Localization HQ extension main window after new project was created

In general, the user interface of the main window contains the following elements:

  • Toolbar
  • Navigation Tree
  • Resource Pane
  • Status Bar
  • Information pane
