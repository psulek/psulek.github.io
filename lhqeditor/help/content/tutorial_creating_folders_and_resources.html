Tutorial Creating Categories and Resources

Two categories (languages, messages) are already created by default. Expand them. Each category contains few default resources:

New Category

To create a new category, select the project root (Strings item) in the navigation tree and click New category button. Name the new category Test and close the dialog. The navigation tree now contains the new category:

New Resource

To create a new resource, select the Test category item in the navigation tree and click New resource button. Name the new resource HelloWorld and close the dialog. The navigation tree now contains the new resource:

Each text resource has its details, see screenshot below. You can edit the individual translations for each language. You can add parameters. You can lock translations. You can change resource state.
