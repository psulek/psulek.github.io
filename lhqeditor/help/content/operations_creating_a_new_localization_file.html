Operations Creating a New Localization File

To create a new localization file, proceed as follows:

  1. In Visual Studio, go to Solution Explorer, right-click on the project item to open the context menu; click Add → New Item. The Add New Item dialog appears.
  2. Select an item which is using LHQ template, e.g. Localization File for Windows Forms (NET Framework, C#, ResX).
  3. New Localization File window appears. Setup the New Localization File parameters and if need be, configure the Code Generator Settings.
