Operations Setting Code Generator

To setup code generator, proceed as follows:

  1. Open LHQ window related to the lhq file for which you want to setup the code generator (double-click the lhq file).
  2. In the Status bar, click on Code Generator label. Code Generator Settings window opens.
  3. Modify the parameters according to your needs. For detailed description of available parameters, see Code Generator Settings [user interface].
  4. Click OK to close the window.
