Default language for new projects

This field defines the primary language offered in your new project.

Default value: English

Country specific checkbox

When unchecked, language code contains two characters, e.g. DE. When checked, language code contains also country code, e.g., EN-US.

Default value: Unchecked

Color theme

You can change between light (white), dark (gray) and automatic color theme. Automatic selection means that the color theme of Visual Studio will be used.

Color theme change of Localization HQ extension is applied automatically.

If you change color theme of Visual Studio (Tools → Options → Environment → Color Theme), Localization HQ extension color theme is applied according to its setting.

Lock translations when resource state is changed to final

When checked, after the resource state is changed to final, the individual translations will be locked.

Default value: Checked

Show hints checkbox

When checked, hints and notes are displayed in the user interface.

Default value: Checked
