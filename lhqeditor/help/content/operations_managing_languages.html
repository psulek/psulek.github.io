Operations Managing Languages

You can perform the following operations on languages:

  • Add language
  • Delete language
  • Edit language
    • Use language code only or both language-country code
    • Set primary language

Add a language

To add a language, proceed as follows:

  1. Click Languages button in the Toolbar. Language Settings window opens.
  2. Click Add button. Add Language dialog appears.
  3. Select language from dropdown.
  4. Modify Country specific checkbox if required.
  5. Modify Set as primary language checkbox if required.
  6. Click OK. The dialog closes. Language is added in the grid.
  7. Click OK to close the Language Settings window.

Now you can edit translations for the added language.

Delete a language

To delete a language, proceed as follows:

  1. Click Languages button in the Toolbar. Language Settings window opens.
  2. Select language you want to delete.

    3. Note: You can remove only one language at a time. Multi-selection is not supported.

  3. Click Remove button. Remove Language dialog appears.

    4. Caution: All translations in this language will be permanently deleted. You might consider backing up your LHQ project file or exporting resources, especially if you use the translations in the future.

  4. Click Yes. The dialog closes. Language is deleted from the grid.
  5. Click OK to close the Language Settings window.

All translations in the removed language were deleted from the LHQ project.

Change language code

To change the language code, proceed as follows:

  1. Click Languages button in the Toolbar. Language Settings window opens.
  2. Select language you want to edit.
  3. Click Edit button. Edit Language dialog appears.
  4. Modify Country specific checkbox.

  5. Click OK. Confirmation dialog appears.

  6. Click Yes. The dialog closes. The language is modified.
  7. Click OK to close the Language Settings window.

Note: If need be, you can also change the language itself. Translations from the previous language will be associated to the new language you selected.

Note: When you change the language code (or even a language as such), and you already have existing generated resx files, you should delete the file which used its previous setting. Set primary language

To change the primary language, proceed as follows:

  1. Click Languages button in the Toolbar. Language Settings window opens.
  2. Select language which you want to set as primary.
  3. Click Edit button. Edit Language dialog appears.
  4. Check Set as primary language checkbox.
  5. Click OK. Confirmation dialog appears.

  6. Click Yes. The dialog closes. The primary language is set.
  7. Click OK to close the Language Settings window.
