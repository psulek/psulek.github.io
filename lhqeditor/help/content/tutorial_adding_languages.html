Tutorial Adding Languages and Translating Resources

You can add more languages to your project.

To add a language, proceed as follows:

  1. Click Languages button in the Toolbar. Language Settings window opens.
  2. Click Add button. Add Language dialog appears.
  3. Select Spanish language from dropdown.
  4. Click OK. The dialog closes. The new language is added in the grid.
  5. Click OK to close the Language Settings window.

Note that number of languages in the project was increased to 4. Now you can edit translations for the added language. Select the HelloWorld resource and edit its translations according to the screenshot below:

Save the project file.

Note that the Test category and the HelloWorld resource (as marked by green rectangle in the screenshot above) do not have red vertical indicator next to them. This means, that translations in all project languages exist for these items. On the other hand, categories Languages and Messages contain some resources, which do not have Spanish translations.
