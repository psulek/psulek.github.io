Operations Deleting an Object

Deleting a Resource

To delete a resource, select it in the navigation tree and do one of the following:

  • Press DEL key.
  • Click Delete button.
  • Right-click to open context menu and click Delete menu item.

A confirmation dialog appears. Click Yes. The selected resource and its translations will be deleted from the project.

Caution: All objects you delete will be removed from the project. You can still use Undo function in Visual Studio after you performed the delete operation. If you delete large amount of resources, or if you perform several delete operations, we recommend to do a backup of your lhq project.

Deleting a Category

To delete a category, select the category folder in the navigation tree and do one of the following:

  • Press DEL key.
  • Click Delete button.
  • Right-click to open context menu and click Delete menu item.

A confirmation dialog appears. Click Yes. The selected category folder and all objects under this category will be deleted from the project.

Deleting Multiple Objects

To delete multiple objects (resource(s) and / or category(ies)), select the objects in the navigation tree and do one of the following:

  • Press DEL key.
  • Click Delete button.
  • Right-click to open context menu and click Delete menu item.

A confirmation dialog appears. Click Yes. The selected resource(s), category folder(s) and all objects under selected category(ies) will be deleted from the project.
