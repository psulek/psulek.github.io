Operations Creating a New Category

To create a new category, proceed as follows:

  1. Select an item in the navigation tree.
  2. Click New category button or use context menu by right-clicking the item and selecting the New category menu item. New category dialog appears.
  3. Enter the category name and click OK. The category is added in the navigation tree and its details are displayed in the Resource pane.
