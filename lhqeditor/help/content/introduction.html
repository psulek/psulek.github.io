Structure of the Web Help

The following high-level topics are present:

  • Introduction - Basic information about the product, symbols and notation, keyboard shortcuts.
  • Installation - How to install Localization HQ extension to Visual Studio.
  • Configuration - How to configure the extension.
  • Tutorial - Example how to start using Localization HQ, creating resources, generating classes, etc.
  • Operations - Detailed steps of operations, the how frame of reference.
  • User Interface - Detailed descriptions of the user interface elements and parameters, default values, purpose (the why frame of reference).

Operations and user interface topics often reference each other to offer more detailed information.
