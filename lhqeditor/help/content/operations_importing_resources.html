Operations Importing Resources

To import resources to the project, proceed as follows:

  1. Click File → Import Resources menu item. Import Resources window opens.

    2. If there are unsaved project changes, a dialog appears first, to save the current project changes:


  2. Select importer from the Available Importers dropdown.
  3. Select files to import.
  4. Modify Allow import of new languages checkbox if required.
  5. Select Import mode option (see Import Resources [user interface]).
  6. Click OK button to start the import.

After the resources are imported, success message appears.


Figure: Resources imported successfully

Imported resources are added to the project according to Import mode setting.
