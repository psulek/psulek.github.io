Operations Creating a New Resource

To create a new resource, proceed as follows:

  1. Select an item in the navigation tree.
  2. Click New resource button or use context menu by right-clicking the item and selecting the New resource menu item. New resource dialog appears.
  3. Enter the resource name and click OK. The resource is added in the navigation tree and its details are displayed in the Resource pane.

Note: If associated code generator does not support resources to be located directly under the root item, you have to create a category first and then create resource(s) under this category.
