Uninstallation Procedure

To uninstall Localization HQ Visual Studio extension, proceed as follows:

  1. In Visual Studio, go to Tools → Extensions and Updates. Find LHQ Editor in the list of installed extensions and click Uninstall.


    Confirmation dialog appears.

  2. Click Yes.


    Visual Studio modification process starts.


  3. Once the modification process finishes, click Close.


  4. An information appears, that changes will be scheduled. Restart Visual Studio.
